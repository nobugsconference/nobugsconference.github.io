This page aims to highlight existing collaborations in software between facilities that have an interest in NOBUGS. Newcomers to the field should get an overview of what is already successfully shared. This is not to stop new developments, but to avoid unnecessary duplication and help direct resources into the right areas for the benefit of the users.

The list does not follow any particular order. To be added to the set of collaborations a software project needs to be actively developed and used by at least two facilities that are represented in the [[International Advisory Committee]] and have been presented at a NOBUGS conference.

DAWN

http://www.dawnsci.org/
Collaboration Partners
  • Diamond Light Source, UK
  • ESRF, France
  • EMBL, EU
 Goals

DAWN (Data Analysis WorkbeNch) is an Eclipse based workbench for doing scientific data analysis. It implements sophisticated support for:

  1. Visualization of data in 1D, 2D and 3D
  2. Python script development, debugging and execution
  3. Workflows for analyzing scientific data calling Python and binary codes

DAWN is not restricted to one scientific domain. It is developed by and for the synchrotron community foremost but has strong overlap with other communities like neutron scattering, photon science and any scientific communities with the above or similar needs.

ICAT

  http://icatproject.org
Collaboration Partners
  • DESY, Germany
  • DLS, UK
  • Elettra, Italy
  • ESRF, France
  • HZB, Germany
  • ISIS, United Kingdom
  • SNS, USA
  • ALBA, Spain
 Goals

ICAT is a data catalogue which aims to support Large Facility experimental data linking all aspects of the research chain from proposal through to publication. It includes a number of components the principal ones being:

ICAT
the metadata catalogue provides a SOAP and RESTful web service interfaces to the underlying database with an easy to use API.
TopCAT
a web based GUI able to search across multiple ICAT instances.
IDS
the ICAT Data Service which provides a uniform interface to data catalogued by ICAT. It can be used to download and upload data and supports a wide range of backend storage via a plugin mechanism.
IJP
the ICAT Job Portal to allow submission of jobs to HPC resources or compute farms where data selection is based on ICAT information.

Mantid

http://www.mantidproject.org
Collaboration Partners * ISIS, Rutherford Appleton Laboratory, UK * ORNL, USA * European Spallation Source, Scandinavia Goals

The Mantid project provides a framework that supports high-performance computing and visualisation of scientific data.

Mantid has been created to manipulate and analyse neutron scattering and muon spectroscopy data, but could be applied to many other techniques.

NeXus

http://www.nexusformat.org/
Collaboration Partners
  • ISIS, Rutherford Appleton Laboratory, UK
  • ImgCIF
  • Lawrence Berkeley Laboratory, USA
  • Los Alamos National Laboratory, USA
  • Helmholtz Zentrum Berlin, Germany
  • Advanced Photon Source, USA
  • Paul Scherrer Institut, Switzerland
  • Australian Nuclear Science and Technology Organisation, Australia
  • Argonne National Laboratory, USA
  • Spallation Neutron Source, USA
  • Diamond Light Source, UK
  • KEK, Japan
  • DESY, Germany
  • JCNS, Germany
 Goals

A common data format for neutron, x-ray and muon science.

NeXus is developed as an international standard by scientists and programmers representing major scientific facilities in Europe, Asia, Australia, and North America in order to facilitate greater cooperation in the analysis and visualization of neutron, x-ray, and muon data.
