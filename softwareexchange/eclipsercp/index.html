Eclipse-RCP Plugins

Many people in the community use the eclipse-rich client platform, eclipse-rcp, in order to construct software. Eclipse-RCP provides for an application framework which can be extended by plugins and many other useful facilities. Such plugins may be reused by other applications or may serve as an example code base for writing own software. This page is dedicated to the exchange of such plugins. You are invited to share information and your plugins through this page.

Rules

  • Everyone can create an account and start adding information to this area.
  • All plugins listed here must be available under some form of free licence: GPL, BSD etc., as source code.
  • For each entry the following information must be provided:
    • Title
    • A description what the plugin does.
    • A link or information where to get the source from. You may leave sources directly on this site if nothing else is available.
    • Information on the licence under which the plugin is released.
    • The author(s) of the plugin, with e-mail address.
  • The NOBUGS International Advisory Committee reserves itself the right to remove entries from this page when these rules are not abided too and the problem cannot be fixed in consultation with the author.
